# Quran SDK Example

This example demonstrates the basic usage of the Quran SDK.

## Running the Example

```bash
dart run example/example.dart
```

## What This Example Shows

1. **Initialize the SDK** - Load all Quran data into memory
2. **Get a specific verse** - Retrieve verse 1:1 with Arabic and English text
3. **Get all Surahs** - List all 114 Surahs
4. **Search verses** - Search for verses containing specific text
5. **Get Juz information** - Access Juz (parts) data
6. **Get page information** - Access Mushaf page data
7. **Get Sajda verses** - Retrieve prostration verses
8. **Get Hizb information** - Access Hizb (quarters) data
9. **Get audio** - Access audio recitation URLs

## Expected Output

The example will print information about:
- Verse 1:1 in Arabic and English
- Total number of Surahs
- Search results for "mercy"
- Juz 1 range
- Page 1 range
- Sajda verses count
- Hizb 1 information
- Available reciters and audio URL
