# Quran SDK Example

A simple Flutter app demonstrating the usage of the Quran SDK package.

## Getting Started

This example shows how to:
- Initialize the Quran SDK
- Get verses and surahs
- Search in the Quran
- Access Juz, Pages, Sajda, and Hizb data
- Get audio recitation information

## Running the Example

```bash
cd example
flutter pub get
flutter run
```

## What You'll See

The app displays various examples of SDK usage including:
- Verse retrieval (Arabic and English)
- Total number of Surahs
- Search functionality
- Juz information
- Page count
- Sajda verses count
- Hizb count
- Available reciters

For more detailed documentation, see the [main README](../README.md).
